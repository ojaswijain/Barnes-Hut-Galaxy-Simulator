RETAIN THE DIRECTORY STRUCTURE TO COMPILE OR RUN

To compile:
*Navigate to the "bh" directory via terminal.
*Assuming you have MingW installed, Run the command mingw32-make.

To run:
*Click on the gui.exe file generated in the bh folder
*The program will shut down automatically after some time due to limitations of SFML screen refresh
*You check also check out main.exe for the data printing, and test.exe for basic SFML introduction
